# Trademark and brand policy

The source code is available under Apache-2.0. That license does not grant a
right to present another product or service as the official `seo` project.

This policy covers the SEO Skills name, seoskills.dev, project artwork, and the
`seo` name when it refers specifically to this package and project. It does not
claim ownership of the generic term "SEO."

## Uses that are fine

- Truthfully saying that software uses, supports, or is based on `seo`.
- Linking to the official repository or seoskills.dev.
- Keeping required attribution and notices in copies or forks.
- Describing a fork as an unofficial fork, provided the distinction is clear.

## Ask first

Open a [GitHub Issue](https://github.com/iannuttall/seo/issues) before using the
project name or artwork as the main identity of a product, company, hosted
service, domain, package, event, or merchandise.

Do not imply endorsement, partnership, or official status. Forks and hosted
versions must use a distinct name and visual identity, and must not create
confusion with the npm package, repository, or seoskills.dev.

Nominative and other uses allowed by applicable law remain allowed.
